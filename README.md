# Visual Search System: 基于计算机视觉的物品查找系统
<br>

## 项目简介
本项目希望使用图像检测和图像相似度匹配技术，实现在大量物品中查找特定物品。  
技术上，项目使用GroundingDino进行目标识别，后端采用Flask实现；  
交互上，项目网站提供“设置目标”与“实时视频”两个页面，用户在上传目标图片后即可看到在实时视频中的物品查找结果。
